# BetaRoom

BetaRoom is a static portfolio demo for an AI-powered beta feedback dashboard.

It helps product builders:

- collect structured beta feedback;
- detect repeated user pain points;
- turn raw feedback into a prioritized product backlog;
- present product decisions in a clean SaaS-style dashboard.

## How to open

Open `index.html` in any modern browser.

## Included pages

The project is a single static HTML file with hash-based routes:

- `#/` — Landing
- `#/dashboard` — Demo product dashboard
- `#/feedback` — Feedback inbox with filters
- `#/backlog` — AI-generated product backlog
- `#/room` — Public beta feedback form

## Tech

- HTML
- CSS
- Vanilla JavaScript
- No external dependencies
- Responsive design

## Portfolio positioning

This project is designed as a GitHub portfolio piece for Junior Product Manager / AI Product Builder roles. It demonstrates product thinking, AI workflow design, dashboard UX, and SaaS-style presentation.
