# Roadmap

## Done in this demo

- Static landing page
- Demo dashboard
- Feedback inbox
- Category filters
- AI backlog simulation
- Public beta feedback form
- LocalStorage feedback capture
- Responsive UI
- GitHub Pages deployment

## Next product iterations

### 1. Real beta rooms

Allow a founder to create multiple beta rooms, each with its own public feedback link.

### 2. LLM-powered feedback clustering

Connect the flow to an LLM API to summarize responses, detect repeated pain points, and generate backlog items.

### 3. Export and sharing

Add export to Markdown, PDF, or CSV so founders can share beta insights with teammates.

### 4. Team workflow

Add comments, task ownership, and status tracking for backlog items.

### 5. Integrations

Send generated tasks to GitHub Issues, Linear, Notion, or Trello.
